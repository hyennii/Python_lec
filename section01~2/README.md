2. input 함수
   input 함수 사용 시, 출력 함수와 다르게 단어를 입력 받음
   출력 함수 안에 입력함수(input) 실행 시, 받은 내용 출력 가능

    ```python
        print("Hello" + input("What is your name?"))
    ```
